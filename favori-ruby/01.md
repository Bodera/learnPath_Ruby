## Interactive Ruby Shell

```ruby
# start irb.
$ irb --simple-prompt
# close irb.
$ exit # you can use quit as an alias.
```
### Your favorite calculator
It's really amazing. Try out math with Interactive Ruby Shell.

Symbol  | Meaning
------------- | -------------
+  | addition
-  | subtraction
*  | multiplication
/  | division

```ruby
$ irb --simple-prompt
>> 3*2
=> 6
>> 4/2
=> 2
>> 3-5
=> -2
>> 3/2
=> 1
```
